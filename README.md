# MERN-DashBoard

###  `BackEnd`
* NodeExpress
* TypeScript
* MongoDB

##
- **`This Project Configurations`**
```
mkdir server
```
```
cd server
```
```
npm init -y
```
```
npm i express body-parser cors dotenv helmet morgan mongoose nodemon 
```
#  
- **`Here You Can Add More Origins To Allow CORS`**
```
npm install cors
```
```
npm install @types/cors
```
#
- **`Not Detected Node Modules`**
```
npm install
```
#
- **`Run Project`**
```
npm start
```
